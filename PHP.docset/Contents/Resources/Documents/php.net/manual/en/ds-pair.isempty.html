Ds\Pair::isEmpty — Returns whether the pair is empty

説明

public bool Ds\Pair::isEmpty ( void )

Returns whether the pair is empty.

パラメータ

この関数にはパラメータはありません。

返り値

Returns TRUE if the pair is empty, FALSE otherwise.

例

例1 Ds\Pair::isEmpty() example

<?php
$a = new \Ds\Pair("a", 1);
$b = new \Ds\Pair();

var_dump($a->isEmpty());
var_dump($b->isEmpty());
?>

上の例の出力は、 たとえば以下のようになります。

bool(false)
bool(true)
