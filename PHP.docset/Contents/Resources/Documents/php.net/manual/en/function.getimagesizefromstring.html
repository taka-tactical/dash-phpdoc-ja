getimagesizefromstring

(PHP 5 >= 5.4.0, PHP 7)

getimagesizefromstring — 画像の大きさを文字列から取得する

説明

array getimagesizefromstring ( string $imagedata [, array &$imageinfo ] )

getimagesize() と似ていますが、 getimagesizefromstring() の最初のパラメータはファイル名ではなく文字列となります。

この関数の挙動の詳細については getimagesize() のドキュメントを参照ください。

パラメータ

imagedata

文字列で表した画像データ。

imageinfo

getimagesize() を参照ください。

返り値

getimagesize() を参照ください。

例

例1 getimagesizefromstring() の例

<?php
$img = '/path/to/test.png';

// ファイルとして開く
$size_info1 = getimagesize($img);

// あるいは、文字列として開く
$data       = file_get_contents($img);
$size_info2 = getimagesizefromstring($data);
?>

getimagesizefromstring function for < 5.4

<?php
   if (!function_exists('getimagesizefromstring')) {
      function getimagesizefromstring($string_data)
      {
         $uri = 'data://application/octet-stream;base64,'  . base64_encode($string_data);
         return getimagesize($uri);
      }
}
?>
