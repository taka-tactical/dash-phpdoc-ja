MongoDB\Driver\WriteError::getInfo

(mongodb >=1.0.0)

MongoDB\Driver\WriteError::getInfo — Returns additional metadata for the WriteError

説明

final public mixed MongoDB\Driver\WriteError::getInfo ( void )

パラメータ

この関数にはパラメータはありません。

返り値

Returns additional metadata for the WriteError, or NULL if no metadata is available.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。
