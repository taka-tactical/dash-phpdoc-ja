ReflectionParameter::isCallable

(PHP 5 >= 5.4.0, PHP 7)

ReflectionParameter::isCallable — パラメータが callable かどうかを返す

説明

public bool ReflectionParameter::isCallable ( void )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

パラメータが callable の場合に TRUE、そうではない場合に FALSE を返します。 失敗した場合は NULL を返します。
