intdiv

(PHP 7)

intdiv — 整数値の除算

説明

int intdiv ( int $dividend , int $divisor )

dividend を divisor で割った整数商を返します。

パラメータ

dividend

割られる数。

divisor

dividend を割る数。

返り値

dividend を divisor で割った整数商。

エラー / 例外

divisor が 0 の場合、DivisionByZeroError 例外がスローされます。dividend が PHP_INT_MIN で divisor が -1 の場合、 ArithmeticError 例外がスローされます。

例

例1 intdiv() の例

<?php
var_dump(intdiv(3, 2));
var_dump(intdiv(-3, 2));
var_dump(intdiv(3, -2));
var_dump(intdiv(-3, -2));
var_dump(intdiv(PHP_INT_MAX, PHP_INT_MAX));
var_dump(intdiv(PHP_INT_MIN, PHP_INT_MIN));
var_dump(intdiv(PHP_INT_MIN, -1));
var_dump(intdiv(1, 0));
?> 
int(1)
int(-1)
int(-1)
int(1)
int(1)
int(1)

function intdiv_1($a, $b){
