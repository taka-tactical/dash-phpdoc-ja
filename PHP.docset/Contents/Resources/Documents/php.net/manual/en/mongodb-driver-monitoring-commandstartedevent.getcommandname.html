MongoDB\Driver\Monitoring\CommandStartedEvent::getCommandName

(mongodb >=1.3.0)

MongoDB\Driver\Monitoring\CommandStartedEvent::getCommandName — Returns the command name

説明

final public string MongoDB\Driver\Monitoring\CommandStartedEvent::getCommandName ( void )

Returns the command name (e.g. "find", "aggregate").

パラメータ

この関数にはパラメータはありません。

返り値

Returns the command name.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。
