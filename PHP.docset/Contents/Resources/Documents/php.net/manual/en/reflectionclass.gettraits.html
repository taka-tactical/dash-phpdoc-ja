ReflectionClass::getTraits

(PHP >= 5.4.0)

ReflectionClass::getTraits — このクラスが使うトレイトの配列を返す

説明

public array ReflectionClass::getTraits ( void )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

トレイト名をキー、トレイトの ReflectionClass のインスタンスを値とする配列を返します。 エラー時には NULL を返します。

User Contributed Notes

Anonymous 16-Jan-2015 01:03
Note that getTraits will NOT return any traits inherited from a parent.

This is currently viewed as the desired behavior.  See https://bugs.php.net/bug.php?id=61554
