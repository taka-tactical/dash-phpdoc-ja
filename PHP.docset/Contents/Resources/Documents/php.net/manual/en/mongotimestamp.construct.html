MongoTimestamp::__construct

(PECL mongo >= 1.0.1)

MongoTimestamp::__construct — 新しいタイムスタンプを作成する

このメソッドを定義している拡張モジュールは非推奨です。 かわりに MongoDB 拡張モジュールを使うべきです。 このメソッドの代替として、以下が使えます。

  • MongoDB\BSON\Timestamp::__construct()

説明

public MongoTimestamp::__construct ([ int $sec = time() [, int $inc ]] )

新しいタイムスタンプを作成します。 パラメータを指定しなければ、現在時刻を使って増分は自動的に提供します。 増分はモジュールが読み込まれたときに 0 に設定され、 このコンストラクタがコールされるたびに ($inc パラメータを渡した場合は除く) ひとつずつ増えていきます。

パラメータ

sec

エポック (1 Jan 1970 00:00:00.000 UTC) からの経過秒数。

inc

増分。

返り値

新しいタイムスタンプを返します。
