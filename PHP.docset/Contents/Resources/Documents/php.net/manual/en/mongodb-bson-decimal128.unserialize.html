MongoDB\BSON\Decimal128::unserialize

(mongodb >=1.2.0)

MongoDB\BSON\Decimal128::unserialize — Unserialize a Decimal128

説明

final public void MongoDB\BSON\Decimal128::unserialize ( string $serialized )

パラメータ

serialized

The serialized MongoDB\BSON\Decimal128.

返り値

Returns the unserialized MongoDB\BSON\Decimal128.

エラー / 例外

  • 引数のパースに失敗した場合に MongoDB\Driver\Exception\InvalidArgumentException をスローします。
  • Throws MongoDB\Driver\Exception\UnexpectedValueException if the properties cannot be unserialized (i.e. serialized was malformed).
  • Throws MongoDB\Driver\Exception\InvalidArgumentException if the properties are invalid (e.g. missing fields or invalid values).
