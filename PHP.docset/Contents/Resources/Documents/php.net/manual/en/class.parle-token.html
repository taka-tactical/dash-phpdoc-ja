The Parle\Token class

(PECL parle >= 0.5.2)

はじめに

This class represents a token. Lexer returns instances of this class.

クラス概要

Parle\Token {
/* Constants */
const integer EOI = 0 ;
const integer UNKNOWN = -1 ;
const integer SKIP = -2 ;
/* プロパティ */
public integer $id ;
public string $value ;
/* メソッド */
}

プロパティ

id

Token id.

value

Token value.

定義済み定数

Parle\Token::EOI

End of input token id.

Parle\Token::UNKNOWN

Unknown token id.

Parle\Token::SKIP

Skip token id.
