Error::getTrace — Gets the stack trace

説明

final public array Error::getTrace ( void )

Returns the stack trace.

パラメータ

この関数にはパラメータはありません。

返り値

Returns the stack trace as an array.

例

例1 Error::getTrace() example

<?php
function test() {
 throw new Error;
}

try {
 test();
} catch(Error $e) {
 var_dump($e->getTrace());
}
?>

上の例の出力は、 たとえば以下のようになります。

array(1) {
  [0]=>
  array(4) {
    ["file"]=>
    string(22) "/home/bjori/tmp/ex.php"
    ["line"]=>
    int(7)
    ["function"]=>
    string(4) "test"
    ["args"]=>
    array(0) {
    }
  }
}
