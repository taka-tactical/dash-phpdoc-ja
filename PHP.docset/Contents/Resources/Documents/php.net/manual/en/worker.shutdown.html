Worker::shutdown

(PECL pthreads >= 2.0.0)

Worker::shutdown — Shutdown the worker

説明

public bool Worker::shutdown ( void )

Shuts down the worker after executing all of the stacked tasks.

パラメータ

この関数にはパラメータはありません。

返り値

Whether the worker was successfully shutdown or not.

例

例1 Shutdown the referenced worker

<?php
$my = new Worker();
$my->start();
/* stack/execute tasks */
var_dump($my->shutdown());

上の例の出力は以下となります。

bool(true)
