GearmanClient::returnCode

(PECL gearman >= 0.5.0)

GearmanClient::returnCode — Get the last Gearman return code

説明

public int GearmanClient::returnCode ( void )

Returns the last Gearman return code.

パラメータ

この関数にはパラメータはありません。

返り値

A valid Gearman return code.
