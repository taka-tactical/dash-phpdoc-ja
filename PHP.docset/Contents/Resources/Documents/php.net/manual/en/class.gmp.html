GMP クラス

(PHP 5 >= 5.6.0, PHP 7)

はじめに

A GMP number. These objects support overloaded arithmetic, bitwise and comparison operators.

注意:

GMP オブジェクトを操作するオブジェクト指向インターフェイスは用意されていません。 手続き型の GMP API を使いましょう。
