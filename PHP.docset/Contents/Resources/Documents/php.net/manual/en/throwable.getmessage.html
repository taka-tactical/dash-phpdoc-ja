Throwable::getMessage

(PHP 7)

Throwable::getMessage — メッセージを取得する

説明

abstract public string Throwable::getMessage ( void )

スローされたオブジェクトに関連づけられたメッセージを返します。

パラメータ

この関数にはパラメータはありません。

返り値

スローされたオブジェクトに関連づけられたメッセージを返します。
