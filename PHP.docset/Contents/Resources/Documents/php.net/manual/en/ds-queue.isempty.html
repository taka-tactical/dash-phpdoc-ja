Ds\Queue::isEmpty

(PECL ds >= 1.0.0)

Ds\Queue::isEmpty — Returns whether the queue is empty

説明

public bool Ds\Queue::isEmpty ( void )

Returns whether the queue is empty.

パラメータ

この関数にはパラメータはありません。

返り値

Returns TRUE if the queue is empty, FALSE otherwise.

例

例1 Ds\Queue::isEmpty() example

<?php
$a = new \Ds\Queue([1, 2, 3]);
$b = new \Ds\Queue();

var_dump($a->isEmpty());
var_dump($b->isEmpty());
?>

上の例の出力は、 たとえば以下のようになります。

bool(false)
bool(true)
